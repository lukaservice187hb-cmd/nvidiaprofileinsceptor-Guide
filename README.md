Step.1 Simply install and unpack
Step.2 open the nvidiaProfileInspector.exe
Step.3 now go to profiles and search for a game (like Fortnite or gta)
Step.4 now you can all nvidia settings change
Step.5 when you have done go to top right and click on Apply changes
